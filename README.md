# URLデコードコピー (URL Decode Copy) - C-Button & Context Menu Edition

Google Chrome上で動作し、表示しているウェブサイトの「タイトル」と「デコード済みのURL」を素早くクリップボードにコピーするための拡張機能です。
ツールバーの専用ボタン（Cバッジ）または右クリックメニュー（コンテキストメニュー）からワンクリックで実行できます。

## 概要

本拡張機能は、ウェブサイトのURLに含まれるエンコーディング（日本語のURLなど）を読みやすくデコードし、さらに不要なトラッキングパラメーターを除去した状態で、ページのタイトルと共にコピーします。

特に、タイトルの取得が難しい特定のFAQサイトやニュースサイトにおいて、最適なテキストを抽出する独自のロジックを搭載しています。

## 主な機能

1. **URLのクリーンアップとデコード**
   - URLエンコードされた日本語などを元の文字に復元します。
   - Googleコミュニティ等で付与される不要な個人パラメーター（`uule` など）を自動的に除去します。

2. **高度なタイトル抽出ロジック**
   - ページ内の `h1` タグだけでなく、サイト構造に合わせて最適な要素からタイトルを取得します。
   - **対応サイト例:**
     - **Epson FAQ:** ページ上部の「主な手順」等のパンくずではなく、具体的な「質問内容」を優先取得。
     - **Yahoo!ニュース:** 記事のメイン見出しを正確に取得。
     - **ESET サポート:** FAQの質問タイトルをピンポイントで取得。
     - **Google Blogspot:** 記事タイトル（h3）に対応。
     - **Google Help** 記事URLの uule パラメーターを削除
    

3. **二段構えのコピー方式**
   - 最新の `Clipboard API` を優先しつつ、セキュリティ制限のあるページでは `textarea` を用いたフォールバック処理を行うため、ほぼ全てのウェブページで確実に動作します。

4. **２つのインターフェース**
   - **Cボタン:** ツールバーのアイコン（Cバッジが表示されます）をクリック。
   - **コンテキストメニュー:** ページ上のどこでも右クリックしてメニューから実行。

## ファイル構成

- `manifest.json`: 拡張機能の定義ファイル。
- `background.js`: 右クリックメニューの生成、ボタンクリックの検知、およびページ内でのコピー処理を制御するメインロジック。

## インストール方法

1. このリポジトリのファイルをすべてダウンロードし、任意のフォルダに展開します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して開きます。
3. 右上の「デベロッパー モード」を **ON** にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、手順1で展開したフォルダを選択します。
5. ツールバーに「C」のバッジがついたアイコンが表示されれば完了です。

## 使い方

### 方法A：ツールバーのボタン
1. コピーしたいページを開きます。
2. ツールバーの「C」バッジがついた拡張機能アイコンをクリックします。
3. 「[タイトル]\n[クリーンなURL]」の形式でクリップボードに保存されます。

### 方法B：右クリックメニュー
1. ページ上の任意の場所で右クリックします。
2. メニューから「タイトルとデコード済みURLをコピー」を選択します。
